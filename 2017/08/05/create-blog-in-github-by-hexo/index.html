hexo在github创建博客

hexo 创建博客方法 (github)

E:\hexo\blog\source_posts( path of *.md)

  1. 安装Node.js
  2. 安装git for windows
  3. 进入“Git Bash”
    $ git config –global user.name “romepeng”
    $ git config –global user.email “romepeng@outlook.com”

  4. 安装Hexo：
    $ npm install hexo-cli -g

    1. cd E: ; mkdir hexo; cd hexo;
    2. hexo init blog
      cd blog
    3. hexo generate
    4. hexo server [http://localhost:4000]
    5. ssh-keygen -t rsa -C "Github的注册邮箱地址"
      check: ssh -T git@github.com

    6. hexo deploy
      need: vim _config.yml

      deploy:
    type:  git
    repo: gir@github.com:romepeng/romepeng.github.io.git
    branch: master

` npm install hexo-deployer-git --save`
  5. new post update
    hexo new "My New Post" hexo clean hexo g hexo s hexo d ( hexo generate --deploy)
